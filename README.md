## Description
A Pokemon TCG Dashboard project for tracking my personal collection, most wanted cards, decks, and buy/sell list. 

This is an evolution of my previous project, Eevee.

## Tech-Stack
- MasterBall-Backend:
    - Django
    - SQLite3
- MasterBall-Frontend:
    - React w/Typescript
